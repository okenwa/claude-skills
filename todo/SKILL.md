# Todo Skill

Personal task management.

## Capabilities

- List current tasks
- Add new tasks
- Mark tasks complete
- Set priorities
- Group by project/context

## Process

1. **Identify action** - list, add, complete, prioritize
2. **Execute** - update task storage
3. **Confirm** - show updated state

## Storage

Tasks stored in project's todo system or Obsidian vault.

## Rules

- Show tasks grouped by status (pending, in progress, done)
- New tasks default to pending
- Keep task descriptions short and actionable
- When listing, show pending first
- Offer to clear completed tasks periodically
